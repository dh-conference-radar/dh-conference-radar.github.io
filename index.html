About the creator

This website is curated by Giulia Grisot as a lightweight public resource for following Digital Humanities conference dates, host organizations, and source updates.

The site favours reviewed, source-linked updates over fully automatic scraping, because DH associations publish conference information in different formats and at different rhythms.
